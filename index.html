Alerta Atizapán

Aplicación movil para socio formador Protección Civil de Atizapán.

Una aplicación para dispositivos android para que las personas de Atizapán puedan ver alertas que Protección Civil publique, estas alertas se ven en un mapa y un "area afectada" para que puedan ver rutas alternas y no exponerse al riesgo. La aplicación tambien cuenta con un manual de instrucciónes para el "antes", "durante" y "despues" de un acontecimiento.
